- ## [vue-cli-plugin-gh-pages](https://github.com/JaZo/vue-cli-plugin-gh-pages#readme)
___



### Install
``` js
vue add gh-pages
```

### Usage
``` js
npm run gh-pages
```

### Options
``` js
// create 'vue.config.js' in root folder //

module.exports = {
    pluginOptions: {
        ghPages: {
            message: 'Updates',
        },
    },
};
```
supports all [gh-pages options](https://github.com/tschaub/gh-pages#options)
